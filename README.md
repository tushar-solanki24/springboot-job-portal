# Spring Boot Job Portal

A Job Portal web application built using **Spring Boot, Thymeleaf, and PostgreSQL**.

## Features

- User Registration & Login
- BCrypt Password Encryption
- Role-based access (Job Seeker / Recruiter)
- Recruiters can post jobs
- Job seekers can search and apply for jobs
- Resume upload
- Saved jobs
- Applicant tracking
- Pagination for job listings

## Tech Stack

- Java
- Spring Boot
- Spring MVC
- Spring Data JPA
- Thymeleaf
- PostgreSQL
- Bootstrap

## How to Run

1. Clone the repository
2. Open in IDE (STS / IntelliJ)
3. Configure database in `application.properties`
4. Run `JobportalApplication.java`
